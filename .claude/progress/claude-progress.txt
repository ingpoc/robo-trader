### Key Finding
Backend uses `python-dotenv`'s `load_dotenv()` which reads .env file directly
into process memory - NOT from parent shell environment. This means:
- Credentials only need to be in .env file
- No need to export in shell startup files
- OAuth tokens auto-saved to .env after authentication
