# Robo Trader Mission

Robo Trader exists to turn a retail trading workflow into an AI-assisted operating system that is:

- safer than ad hoc manual execution
- more observable than a black-box bot
- paper-trading-first until execution, risk, and monitoring behavior are trustworthy

The product intent in this repo is not just "place trades." It is to coordinate analysis, research, approvals, execution paths, and operator visibility across one backend and one UI.

## What This Repo Is Optimizing For

- Transparent AI reasoning
- Persistent operational state
- Queue-backed and event-driven workflows
- Stronger paper-trading execution before broader live-trading claims
- A UI that acts as an operator console, not just a chart surface
