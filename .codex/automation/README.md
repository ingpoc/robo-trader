# Repo Automation Specs

This directory holds repo-owned automation prompt specs for Robo Trader.

Use these files as the project source of truth for automation behavior,
guardrails, and output requirements.

Keep in mind:

- the runnable Codex automation may still be installed outside the repo
- the prompt and guardrails should be maintained here first
- any installed automation should mirror the current repo spec instead of
  inventing its own prompt

Current specs:

- `autonomous-paper-trader.prompt.md`
